# **App Name**: Мой Остео

## Core Features:

- Управление Видео-Библиотекой: Позволяет врачам создавать и управлять библиотекой видео-упражнений с деталями, такими как описания, зоны и ограничения. Возможность импорта метаданных видео, таких как название, превью и продолжительность, используя URL видео с Rutube / YouTube.
- Управление Пациентами: Позволяет врачам создавать и управлять профилями пациентов, назначать видео-упражнения и отслеживать активность.
- Персонализированные Видео-Назначения: Позволяет врачам назначать конкретные видео пациентам, основываясь на их потребностях, с возможностью установки продолжительности, порядка и комментариев.
- Создание Шаблонов: Позволяет врачам создавать повторно используемые шаблоны для общих состояний или этапов лечения, чтобы быстро назначать видео пациентам.
- Доступ Пациента: Предоставляет пациентам доступ к персональной странице, содержащей назначенные видео-упражнения и инструкции, доступной по уникальной ссылке.
- Логирование Активности: Регистрирует, какие видео открывали пациенты, чтобы предоставить врачу информацию о прогрессе пациента.
- Умный Генератор Комплексов (Tool): Предлагает набор видео-упражнений для пациента в соответствии с областью, целью и уровнем, используя генеративный ИИ, чтобы помочь врачу.

## Style Guidelines:

- Основной цвет: Мягкий бирюзовый (#74C69D), вдохновленный ощущением исцеления, здоровья и благополучия. Используется в качестве успокаивающего, но свежего фона, чтобы смягчить любые чувства беспокойства или тревоги по поводу травмы. (Примечание: не тот оттенок бирюзового, которого избегают модели; намеренно выбран вариант с преобладанием зеленого).
- Цвет фона: Очень светло-зеленый (#F0F9F4), чтобы создать ощущение чистоты и клинической атмосферы, но все же находящийся в том же диапазоне оттенков, что и основной цвет.
- Цвет акцента: Бледный, приглушенный синий (#94D2BD). Мягко контрастирует с зеленой цветовой схемой, чтобы выделить важные действия и информацию.
- Шрифт для тела и заголовков: 'PT Sans', современный гуманистический шрифт без засечек, который выглядит клиническим, но доступным.
- Используйте четкие и простые иконки для представления различных зон тела и типов упражнений. Рассмотрите линейные иконки для современного вида.
- Поддерживайте чистый и организованный макет, уделяя приоритетное внимание простоте навигации как для врачей, так и для пациентов. Трехколоночный макет для назначения видео имеет важное значение.
- Используйте тонкие переходы и анимацию для улучшения пользовательского опыта, например, при назначении видео или отметке упражнений как завершенных.
